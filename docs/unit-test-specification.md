# 単体テスト仕様書

## 1 概要

書籍管理APIに対する単体テストを実施し、各機能が要件通り動作することを確認する。

## 2 テスト対象

| 機能 | API                          | 
|--|------------------------------|
| 著者登録 | POST /authors                |
| 著者更新 | PUT /authors/{authorId}      |
| 書籍登録 | POST /books                  |
| 書籍更新 | PUT /books/{bookId}          |
| 著者別書籍検索 | GET /authors/{authorId}/books |

## 3.1 著者登録

| No | テスト内容 | 入力値            | 期待結果                                      | 検証方法  | 検証結果 |
|----|-----------|----------------|-------------------------------------------|-------|------|
| A-01 | 正常登録 | 正常データ          | 著者情報が登録される                                | Junit | OK   |
| A-02 | 名前未指定 | name=null      | 400 Bad Request が返却される                    | Postman | OK   |
| A-03 | 生年月日未指定 | birthDate=null | 400 Bad Request が返却される                    | Postman | OK   |
| A-04 | 生年月日が未来日 | 現在日の翌日         | {"message": "生年月日は現在日以前を入力してください。"}が返却される | Junit | OK   |
